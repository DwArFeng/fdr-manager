# fdr-manager

作者的开源项目 `flight-data-record`（简写为 `fdr`） 的管理器，提供了一个可以提供维护和查询的web ui界面。

[什么是fdr？它到底是干什么的？](https://github.com/DwArFeng/fdr)

## 界面

- 实体管理
  - 数据点管理
  - 过滤器管理
  - 触发器管理
- 数据支持
  - 过滤器支持
  - 触发器支持
  - 映射器支持
- 数据查询
  - 实时数据查询
  - 持久数据查询
  - 过滤数据查询
  - 触发数据查询
